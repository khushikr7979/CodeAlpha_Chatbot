## Description
This is a basic rule-based chatbot built using Python as part of the CodeAlpha Internship program.

## Features
- Responds to greetings like hello, hi, hey
- Answers basic questions
- Exits when the user types 'bye'

## Technologies Used
- Python

## How to Run
1. Open terminal in the project folder
2. Run the command:
   python chatbot.py
